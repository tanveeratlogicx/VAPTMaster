# VAPT Master User Guide

## Introduction
Welcome to VAPT Master, your ultimate security companion for WordPress. This build has been customized specifically for your domain to protect against VAPT and OWASP vulnerabilities.

## Installation
1. Upload the `vapt-master-client.zip` via the WordPress admin (Plugins > Add New > Upload).
2. Activate the plugin.
